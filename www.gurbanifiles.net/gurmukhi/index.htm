All Words from Siri Guru Granth Sahib, Gurmukhi
This document has all the words from Siri Guru Granth Sahib in Gurmukhi and these words are arranged in an alphabetical order according to the Gurmukhi alphabet. The words in this document number approximately 30,000. This document can be very useful in doing word searches of Siri Guru Granth Sahib files. For example, before doing a word search one can look at all the variations of a word or similarities in the words. One can also copy the word for pasting in the "Find" dialogue window to make sure that the spellings are correct for the search. This file is in Unicode Gurmukhi font.
